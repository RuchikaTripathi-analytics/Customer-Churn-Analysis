# Customer Churn Analysis

## Overview

This project analyzes customer churn behavior using Python and Power BI. The objective is to identify factors influencing customer attrition and provide actionable business insights.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI

## Key Performance Indicators (KPIs)

* Total Customers: 7043
* Active Customers: 5174
* Churned Customers: 1869
* Churn Rate: 26.58%

## Key Insights

* Month-to-month contract customers show the highest churn.
* Customers using electronic check payment methods have higher churn rates.
* Fiber optic internet users churn more frequently than DSL users.
* Churned customers have higher monthly charges.
* Churned customers have significantly lower average tenure.

## Dashboard Features

* KPI Cards
* Churn by Contract Type
* Churn by Payment Method
* Churn by Internet Service
* Average Monthly Charges by Churn
* Average Tenure by Churn
* Interactive Filters and Slicers
